# LostFound+ Mobile App

LostFound+ is a JavaScript Expo and React Native project. The app helps students report lost and found items, save important reports, and manage item information clearly on mobile.

## Devloped By
- Keval Pathak

## Main Features
- Multi-screen mobile app with tab + stack navigation
- Lost & found reporting system
- AsyncStorage-based persistence
- REST API integration
- Full CRUD operations
- Loading, error, and empty state handling

## Screens
- Home Dashboard
- Reports List
- Create / Edit Report
- Item Details
- Saved Reports
- Profile / Contribution Summary

## Installation
```bash
npm install
npx expo start
```

## How to run on Android Emulator
```bash
npx expo start
```
Then press `a` in the terminal or click Run on Android device/emulator.
