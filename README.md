# LostFound+ Mobile App

LostFound+ is a JavaScript Expo and React Native project. The app helps students report lost and found items, save important reports, and manage item information clearly on mobile.

## Devloped By
- Keval Pathak

## Main Features
- 4+ screens with bottom tabs and nested stack navigation
- Parameterized detail screen using `itemId`
- AsyncStorage persistence for lost/found reports
- Create, update, delete, display, and save reports
- Loading, error, and empty states
- Remote REST API integration using Random User API

## Screens
- Home Dashboard
- Reports List
- Create / Edit Report
- Item Details
- Saved Reports
- Profile / Contribution Summary

## Installation
```bash
npm install
npx expo start
```

## How to run on Android Emulator
```bash
npx expo start
```
Then press `a` in the terminal or click Run on Android device/emulator.
